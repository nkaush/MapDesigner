PROJECT TITLE: Subway Map Creation
PURPOSE OF PROJECT: Easily create a subway map with 3 different train lines
VERSION or DATE: 11/8/18
HOW TO START THIS PROJECT: Run the main method in the Starter Class to create the map
AUTHORS: Neil Kaushikkar
USER INSTRUCTIONS: Click wherever you want in the window to add a station. Switch between lines
                   using the buttons in the top left corner of the window.
