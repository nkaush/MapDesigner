PROJECT TITLE: Map Designer
PURPOSE OF PROJECT: Easily create an interactive map.
VERSION or DATE: 11/8/19
HOW TO START THIS PROJECT: Run the main method in the Starter Class to create the map
AUTHORS: Neil Kaushikkar
USER INSTRUCTIONS: Click wherever you want in the window to add a node. Switch between paths
using the buttons in the top left corner of the window.

Initially designed to create subway maps, but can be used to create any interactive map.
